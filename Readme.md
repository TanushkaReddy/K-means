## K-means
# Smartphone User Segmentation using K-Means

## Overview
This project uses the K-Means Clustering algorithm to group smartphone users based on their usage behavior and identify different user segments.

## Features Used
- Daily Screen Time
- Social Media Hours
- Gaming Hours
- Productivity App Usage
- Data Usage
- Battery Drain
- Apps Installed
- Notifications Per Day

## Steps Performed
1. Data Loading
2. Data Preprocessing
3. Feature Scaling
4. K-Means Clustering
5. PCA Visualization
6. Cluster Analysis

## Model Evaluation
- Silhouette Score: 0.52

## Output
The model segments users into different groups such as:
- Heavy Users
- Gamers
- Productivity Users
- Low Usage Users
